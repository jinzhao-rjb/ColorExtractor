# 图片颜色提取工具

一个简单易用的Web应用，用于从上传的图片中提取主要颜色。

## 功能特点

- 支持多种图片格式上传（JPG、PNG、WEBP等）
- 提供多种颜色提取算法：K-means聚类、最频繁颜色、主色调提取、中位切分法、颜色分层提取
- 可自定义提取的颜色数量
- 支持HEX和RGB两种颜色格式显示
- 深色/浅色模式切换
- 点击颜色可复制到剪贴板

## 本地开发

1. 克隆仓库
2. 直接在浏览器中打开 `index.html` 即可运行

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)

## 部署

项目支持GitHub Pages一键部署：
1. 将代码推送到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择主分支作为发布源
